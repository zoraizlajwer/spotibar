Rainmeter Skin

Control Spotify from your Taskbar

A clean, interactive media controller skin for Rainmeter. Launch Spotify directly from your taskbar, view beautiful playback summaries, and enjoy continuous dynamic themes.

View Project on GitHub Install Guide

Key Features

A minimalist skin tailored perfectly to look like a native Windows 11 taskbar tool.

01

Dynamic Accents

Both the circular progress bar around the album art and the song title text dynamically extract and adopt an accent color matching the current track's album cover.

02

Intelligent UI Layers

Hovering over the album cover reveals context menus (Controls Panel or Song Info Panel). The skin automatically minimizes back to just a Spotify icon if the media application isn't active.

03

Spotify Web Support

Not using the desktop application? Enjoy full functionality inside your web browser by pairing the skin layout with the WebNowPlaying browser extension.

How to Install & Optimize

Follow these structured configuration steps to anchor Spotibar smoothly down inside your Windows environment.

1. Setup Core Requirements

Ensure you have the standard Spotify Desktop application installed. You also need Rainmeter 4.5.18 (or newer) running on your PC. To use Spotify Web, download the WebNowPlaying extension.

2. Download and Load Spotibar

Download the official package release from the Spotibar Releases on GitHub. Open Rainmeter, hit refresh, and extract/load the layout.

3. Mandatory Skin Layout Settings

To keep your navigation locked precisely onto the taskbar frame, configure the loaded skin parameters with these settings:

  • Position: Change value to Stay topmost
  • Draggable: Uncheck this option
  • Click through: Uncheck this option

Media Playback Triggers

The player handles deep functions instantly. Use Previous Track, Play/Pause, Next Track, or toggle Repeat variables using the dedicated visual icons.

Smart Gestures & Clicks

  • Open Spotify Window: Click on the tracking album cover or the track description details panel.
  • Volume Adjustment: Scroll your mouse wheel directly over the skin frame, or use a responsive two-finger swipe gesture on a laptop trackpad.
  • Mute / Unmute: Simple Right-Click anywhere over the active skin.

Important Running Note

Always minimize your core Spotify window to the system tray rather than fully closing out the application box so that data handles stay live.

Get the Windows 11 Floating Look

To achieve the clean, rounded taskbar segmentation shown in the preview image, follow this optional layout process:

1. Download Windhawk

Install the open-source mod runner Windhawk on your system.

2. Install Styler Engine

Inside the Windhawk app engine catalog, search for and activate the Windows 11 Taskbar Styler mod.

3. Apply the Matter Theme

Select the preset theme option named Matter to break your long taskbar down into the clean modules shown in the preview.
